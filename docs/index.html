Api

Класс Api
Отвечает за отправку запросов на сервер и проверку полученных ответов

Api
Parameters
Object (Object) принимает в конструктор объект { URLs, headers, token }
Name Description
Object.URLs Object объект, содержащий адреса для отправки запросов
Object.headers Object объект, содержащий заголовки запросов
Object.token String токен для доступа на сервер
Example
const api = new Api({
URLs: {
  baseURL: 'https://mesto...',
  cardsURL: 'https://mesto.../cards/',
  userURL: 'https://mesto.../users/me',
  likesURL: 'https://mesto.../cards/likes/',
  avatarURL: 'https://mesto.../users/me/avatar'
},
headers: {
  "authorization": '<Ваш код авторизации>'
},
token: '<Ваш токен>'
});

Cards

Класс Card
Создает экземпляр элемента (карточки)
Принимает в конструктор объект с данными для создания карточки и коллбэками, селекторы элементов карточки, идентификаторы состояния

Card
Parameters
Object (Object) : - объект с основными данными карточки и коллбэками
Name Description
Object.data Object { title, link, id } - объект с данными карточки, необходимыми для ее создания
Object.data.title String название карточки
Object.data.link String ссылка на изображение карточки
Object.data.id String идентификатор карточки
Object.handleCardClick Function обработчик клика по изображению карточки
Object.handleClickDeleteIcon Function обработчик клика по иконке удаления карточки
Object.setSubmitAction Function коллбэк, определяющий функцию-обработчик при подтверждении удаления карточки
Object.handleClickLikeIcon Function обработчик клика по иконке "лайк"
cardTemplateSelector (String) селектор шаблона элемента карточки
cardElementsSelectors (Object) : - объект, содержащий селекторы элементов карточки
Name Description
cardElementsSelectors.cardSelector String селектор карточки
cardElementsSelectors.deleteIconSelector String селектор иконки удаления
cardElementsSelectors.cardImageSelector String селектор изображения карточки
cardElementsSelectors.likeIconSelector String селектор иконки "лайк"
cardElementsSelectors.cardTitleSelector String селектор элемента названия карточки
cardElementsSelectors.isLikedModifier String модификатор, окрашивающий "лайк" в черный цвет, если "лайк" в активном состоянии
cardElementsSelectors.likeCounterSelector String селектор счетчика "лайков"
cardElementsSelectors.delButtonEnabledSelector String селектор активного состояния иконки удаления
isOwner (Boolean) идентификатор владельца:
  true - если пользователь является владельцем карточки,
  false - если пользователь не является владельцем карточки
isLiked (Boolean) идентификатор состояния кнопки "лайк":
  true - если "лайк" нажат,
  false - если "лайк" не нажат
likesQuantity (Number) число - количество "лайков", поставленных пользователями данной карточке

FormValidator

Класс FormValidator
Определяет логику валидации форм в модальных окнах приложения
Для каждой формы создается свой экземпляр класса FormValidator
Класс имеет один публичный метод, запускающий валидацию - enableValidation()
Класс принимает в конструктор объект настроек config и DOM-ноду формы, для которой необходимо запустить валидацию

FormValidator
Parameters
config (Object) объект настроек валидации, содержит селекторы элементов и классы, необходимые для валидации
Name Description
config.inputSelector String селектор инпута формы, общий для всех инпутов
config.submitButtonSelector String селектор кнопки отправки формы (submit)
config.inactiveButtonClass String css класс, устанавливающй неактивное состояние кнопки submit
config.inputErrorClass String css класс, устанавливающй невалидное состояние инпута
config.errorClass String css класс, устанавливающй активное состояние элемента, содержащего текст ошибки
formNode (HTMLElement) DOM-нода формы, для которой необходимо запустить валидацию
Instance Members
▸ enableValidation()

Публичный метод
Включает валидацию формы
Включение валидации производится один раз.
Далее валидация осуществляется постоянно при взаимодействии пользователя с формой

enableValidation()
Example
const config = {
 inputSelector: '.someSelector', 
 submitButtonSelector: '.someSelector',
 inactiveButtonClass: 'someClass',
 inputErrorClass: 'someClass',
 errorClass: 'someClass',
};
const editForm = document.querySelector('.someFormSelector');
const editFormValidator = new FormValidator(config, editForm);
editFormValidator.enableValidation();
