Module: Section

Класс Section
Отвечает за отрисовку элементов на странице Принимает в конструктор объект с данными для создания и отрисовки элементов
Parameters:
Name Type Description
Object Object { items, renderer } - объект с данными для отрисовки элементов на странице
Properties
Name Type Description
items Array Массив объектов, содержащих данные, для создания элементов
renderer function Коллбэк, создающий элемент и размещающий созданный элемент на странице.
Принимает на вход объект item с данными для создания элемента.
containerSelector String Селектор контейнера, в который будут размещаться элементы

Example
const container = new Section({
  items: [{object1}, {object2}, {object3}],
  renderer: someCallback(item)               //где item - объект с данными для создания элемента
}, '.someSelector');                         //селектор контейнера
