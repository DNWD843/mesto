Module: Card

Класс Card
Создает экземпляр элемента (карточки)
Принимает в конструктор объект с данными для создания карточки и коллбэками, селекторы элементов карточки, идентификаторы состояния
Parameters:
Name Type Description
Object Object объект с основными данными карточки и коллбэками
Properties
Name Type Description
data Object { title, link, id } - объект с данными карточки, необходимыми для ее создания
Properties
Name Type Description
title String название карточки
link String ссылка на изображение карточки
id String идентификатор карточки
handleCardClick function обработчик клика по изображению карточки
handleClickDeleteIcon function обработчик клика по иконке удаления карточки
setSubmitAction function коллбэк, определяющий функцию-обработчик при подтверждении удаления карточки
handleClickLikeIcon function обработчик клика по иконке "лайк"
cardTemplateSelector String селектор шаблона элемента карточки
cardElementsSelectors Object объект, содержащий селекторы элементов карточки
Properties
Name Type Description
cardSelector String селектор карточки
deleteIconSelector String селектор иконки удаления
cardImageSelector String селектор изображения карточки
likeIconSelector String селектор иконки "лайк"
cardTitleSelector String селектор элемента названия карточки
isLikedModifier String модификатор, окрашивающий "лайк" в черный цвет, если "лайк" в активном состоянии
likeCounterSelector String селектор счетчика "лайков"
delButtonEnabledSelector String селектор активного состояния иконки удаления
isOwner Boolean идентификатор владельца:
 true - если пользователь является владельцем карточки,
 false - если пользователь не является владельцем карточки
isLiked Boolean идентификатор состояния кнопки "лайк":
 true - если "лайк" нажат,
 false - если "лайк" не нажат
likesQuantity Number число - количество "лайков", поставленных пользователями данной карточке

Example
const cardElementsSelectors = {
  cardSelector: '.someSelector',
  deleteIconSelector: '.someSelector',
  cardImageSelector: '.someSelector',
  likeIconSelector: '.someSelector',
  cardTitleSelector: '.someSelector',
  isLikedModifier: 'someModifier',
  likeCounterSelector: '.someSelector',
  delButtonEnabledSelector: '.someSelector'
};

const card = new Card({
  data: someCardData,
  handleCardClick: someCallback,
  handleClickDeleteIcon: someCallback,
  setSubmitAction: someCallback,
  handleClickLikeIcon: someCallback
},
  cardTemplateSelector, cardElementsSelectors, isOwner, isLiked, likesQuantity
);
