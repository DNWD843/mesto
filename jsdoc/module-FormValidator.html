Module: FormValidator

Класс FormValidator
Определяет логику валидации форм в модальных окнах приложения
Для каждой формы создается свой экземпляр класса FormValidator
Класс имеет один публичный метод, запускающий валидацию - enableValidation()
Класс принимает в конструктор объект настроек config и DOM-ноду формы, для которой необходимо запустить валидацию
Parameters:
Name Type Description
config Object объект настроек валидации, содержит селекторы элементов и классы, необходимые для валидации
Properties
Name Type Description
inputSelector String селектор инпута формы, общий для всех инпутов
submitButtonSelector String селектор кнопки отправки формы (submit)
inactiveButtonClass String css класс, устанавливающй неактивное состояние кнопки submit
inputErrorClass String css класс, устанавливающй невалидное состояние инпута
errorClass String css класс, устанавливающй активное состояние элемента, содержащего текст ошибки
formNode HTMLElement DOM-нода формы, для которой необходимо запустить валидацию

Methods

enableValidation()

Публичный метод
Включает валидацию формы
Включение валидации производится один раз.
Далее валидация осуществляется постоянно при взаимодействии пользователя с формой
